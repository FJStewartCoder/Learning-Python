# Task
Create a loop.
In the loop, ask the user if they want to add a fact to the file, read the facts or quit.

If add fact, ask them for a fact and append it to the file.  
If read facts, read all of the facts and output them.  
If quit, break the loop.  

Try to create a function for each menu option.

Assume the file exists. We can learn later how to handle errors such as these ones.

Create an extra option to clear the file.
