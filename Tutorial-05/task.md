# Task
Create a list of fruits. The list will contain the strings "Apple", "Orange", "Banana" and "BOB".  
Add these options as many times and in any order that you like.

Create a counter and set the value to 0.  

Loop through each of these options. If the fruit is "BOB", end the loop.  
If the fruit is not "BOB", output the name of the fruit.  
If the fruit is not "Apple", loop back to the start of the loop.  
Finally, add 1 to the counter (in the loop).  

Output the result of the counter.  
The result should the number of "Apple"s before encountering "BOB".  

# Bonus Task
Create a counter and set the value to 0.

Create an infinite loop.  
In the loop, add one to the counter and output the number.  
If the number is ever greater than or equal to 23, leave the loop.
