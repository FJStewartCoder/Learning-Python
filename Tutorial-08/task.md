# Task
Create two variables called "username" and "password" (better even if you put them in a dictionary).  

Loop three times and in the loop get the user to enter a username and password.  
Store these somewhere.

If the username and password that the user enters is the same as the inital variables that you made,
print a hello message and exit the loop.

If the user is incorrect, print an error message
