# Task
Create a dictionary in the global scope with each of the characters as keys and corresponding Scrabble scores as values.  
Create a function that takes a word and multiplier as arguments. The multiplier should have a default value of 1.  

Convert the word to lowercase before getting the score.  
Get the score for each letter using the dictionary. You should first check if the letter is in the dictionary.
The function will get the Scrabble score of the word and return the score multiplied by the multiplier.  

Call the function using the values below:  
- Jason, 1
- Timmy, 2
- Bob and James, 5

# Bonus Task
Experiment with making some functions using *args and **kwargs.
